formurlator v0.1.0

formurlator is a JavaScript library meant to synchronize a set of DOM elements with a query string in the URL.

formurlator uses oojs, a JavaScript Object-Oriented library by the Wikimedia Foundation. A standalone version that includes oojs is available for use.

You can read the documentation, play around with demos, or contribute to the library on GitHub.
