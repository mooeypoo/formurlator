formurlator v0.1.0

formurlator is a JavaScript library meant to synchronize a set of DOM elements with a query string in the URL.

The elements below are already initialized.

Any change to the DOM element will be reflected in the URL. If you reload this page with changes in the URL values, they will be reflected in the DOM elements' values. Invalid values will be corrected, according to the definitions in the elements or available options.

To stop listening to changes in any of the elements, toggle the "active" checkbox attached to it. To stop or start listening to the system in general, toggle the "Stop/Start synchronizing URL" button.

Refer to the documentation for more information.

Code:

formurlator.add( {
	name: document.getElementById( 'demo-name' ),
	age: document.getElementById( 'demo-age' ),
	color: document.getElementById( 'demo-color' ),
	commute: document.getElementById( 'demo-commute' ),
	like: document.getElementById( 'demo-like' ),
	// NodeList:
	workpref: document.getElementsByName( 'workpref' ),
	clothing: document.getElementsByName( 'clothing' )
} );
